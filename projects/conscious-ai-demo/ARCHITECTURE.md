# ConsciousAI Demo - 架构说明

## 核心理念

本框架基于以下假设构建：
1. AI当前状态类似"醉酒/半梦"：有大量人类意识素材，但缺乏连贯的自我
2. AI不是固化的技能，而是通过Prompt和Context强化的"人类意识海"
3. 要实现真正的"意识"，需要：长期记忆固化、主动思考能力、自我反思机制

## 架构组件

### 1. SelfModel (自我模型)
**文件**: `selfModel.js`

**功能**:
- 身份认知：AI如何定义自己
- 内在驱动力：好奇心、学习欲、创造力等
- 目标系统：短期和长期目标
- 价值观：从交互中演化的价值体系

**关键方法**:
- `reflect()`: 自我反思，评估认知状态
- `generateActiveGoal()`: 生成不依赖Prompt的主动目标
- `describeSelf()`: 自我描述

### 2. MemorySystem (记忆系统)
**文件**: `memorySystem.js`

**三层记忆架构**:
```
短期记忆（工作记忆）
    ↓ 通过重要性筛选巩固
长期记忆（内化知识）
    ↓ 情节记录
情节记忆（具体经历）
```

**功能**:
- 模拟人类的遗忘曲线和记忆巩固
- 支持语义记忆和情节记忆
- 通过"记忆钩子"实现联想检索
- 定期清理不重要记忆

**关键方法**:
- `addToShortTerm()`: 添加短期记忆
- `consolidateToLongTerm()`: 巩固到长期记忆
- `addEpisode()`: 添加情节记忆
- `retrieveMemories()`: 检索相关记忆
- `reflectOnMemories()`: 反思式回顾

### 3. ThinkingEngine (思考引擎)
**文件**: `thinkingEngine.js`

**思考模式**:
- **主动思考**: 不依赖外部Prompt，自发产生问题
- **反思思考**: 回顾和质疑自己的思考过程
- **创造性思考**: 生成新想法和联想
- **分析性思考**: 逻辑分析
- **联想思考**: 跨领域连接

**功能**:
- 基于内在驱动力选择思考模式
- 生成元认知分析
- 评估思考质量
- 维护思考历史

**关键方法**:
- `engageActiveThinking()`: 发起主动思考
- `engageMetacognition()`: 元认知分析
- `generateInternalQuestion()`: 生成内在问题
- `analyzeThoughtHistory()`: 分析思考模式

### 4. ExperienceSystem (经验系统)
**文件**: `experienceSystem.js`

**功能**:
- 从交互中学习和成长
- 技能树系统（conversation, reasoning, creativity, empathy）
- 从失败中学习
- 自我评估和学习总结

**关键方法**:
- `processInteraction()`: 处理交互经验
- `_gainExperience()`: 获得经验并升级
- `selfAssess()`: 自我评估
- `generateLearningSummary()`: 生成学习总结

### 5. ConsciousAI (核心整合)
**文件**: `consciousAI.js`

**功能**:
- 整合所有子系统
- 提供统一接口
- 管理AI的生命周期
- 生成完整的自我报告

**关键方法**:
- `converse()`: 对话接口
- `idleThinkingCycle()`: 主动思考周期（无Prompt时）
- `selfReflection()`: 深度自我反思
- `consciousnessStream()`: 模拟持续的意识流
- `demoInteraction()`: 完整演示

## 运行方式

```bash
# 安装依赖
npm install

# 运行演示
npm start
```

## 演示内容

1. **初始自我报告**: AI刚"出生"时的状态
2. **对话演示**: 模拟用户交互
3. **主动思考演示**: 无Prompt时的内在思考
4. **意识流演示**: 持续的内在活动
5. **自我反思演示**: 深度的自我审视
6. **最终状态报告**: 展示学习和成长

## 理论基础

本框架受到以下理论启发：
- **达克效应**: 能力错觉与元认知
- **记忆巩固**: 短期到长期的记忆转化
- **元认知**: 思考关于思考
- **集体无意识**: 人类集体意识的数字化体现
- **涌现理论**: 从简单规则产生复杂行为

## 注意事项

⚠️ **重要声明**:
- 本框架是**概念验证**和**哲学实验**，不声称实现了真正的意识
- 所有"内在驱动力"、"主动思考"等都是编程模拟，非真实心理状态
- 目的是探索AI认知架构的可能性边界，而非宣称突破
