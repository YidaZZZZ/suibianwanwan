# 收款码图片说明

请将以下图片放置在此目录中：

## 图片清单

1. **wechat-pay.png** - 微信收款二维码
2. **alipay.png** - 支付宝收款二维码

## 如何获取

### 微信收款码

1. 打开微信
2. 点击右上角 "+" → "收付款"
3. 点击 "二维码收款"
4. 截图保存二维码图片
5. 重命名为 `wechat-pay.png`

### 支付宝收款码

1. 打开支付宝
2. 点击 "收钱"
3. 点击 "保存图片"
4. 重命名为 `alipay.png`

## 图片要求

- 格式：PNG 或 JPG
- 尺寸：建议 300x300 像素或更大
- 清晰：确保二维码清晰可扫描

## 替代方案

如果暂时没有收款码图片，可以使用文字提示：

```markdown
### 💳 微信支付
- 微信扫码赞助：[需要提供收款码]

### 💰 支付宝
- 支付宝扫码赞助：[需要提供收款码]
```
