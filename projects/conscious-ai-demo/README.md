# Conscious AI Demo

基于"人类意识海"假设的AI意识模拟框架

## 核心理念

1. **长期记忆系统** - 不只是临时上下文，而是持久化的个人经验
2. **主动思考引擎** - 不依赖Prompt的内在驱动
3. **自我反思层** - 监控和评估自己的认知过程
4. **持续学习机制** - 从交互中不断成长

## 架构

```
ConsciousAI (意识主体)
├── SelfModel (自我模型)
├── MemorySystem (记忆系统)
│   ├── ShortTermMemory (短期记忆)
│   ├── LongTermMemory (长期记忆)
│   └── EpisodicMemory (情节记忆)
├── ThinkingEngine (思考引擎)
│   ├── ActiveThinking (主动思考)
│   ├── ReflectiveThinking (反思思维)
│   └── GoalSystem (目标系统)
└── ExperienceSystem (经验系统)
    ├── Learning (学习)
    ├── Evaluation (评估)
    └── Adaptation (适应)
```

## 运行

```bash
npm install
npm start
```
