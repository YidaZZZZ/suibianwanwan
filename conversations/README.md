# Conversations - 临时对话目录

## 📁 目录说明

此目录用于存放临时对话产生的文件、代码片段和实验性内容。

---

## 🗂️ 目录结构

```
conversations/
├── 2026-03-31/          # 按日期归档
│   ├── experiment-1.js  # 实验代码
│   ├── test-demo.html   # 测试文件
│   └── notes.md         # 对话笔记
├── 2026-04-01/
│   └── ...
└── README.md
```

---

## 📝 使用规范

### 文件命名

- **日期格式**: `YYYY-MM-DD`
- **实验文件**: `experiment-{N}.{ext}`
- **测试文件**: `test-{name}.{ext}`
- **笔记文件**: `notes.md`

### 归档周期

- **每日对话**: 当天创建日期文件夹
- **代码实验**: 放在对应日期文件夹
- **有价值内容**: 迁移到 `projects/` 目录

### 清理规则

1. **每月清理**: 删除过期的临时文件
2. **重要代码**: 迁移到项目目录
3. **保留笔记**: 对话记录保留

---

## 🔄 内容流转

```
临时对话文件
    ↓
是否需要长期维护？
    ├─ 是 → 移动到 projects/
    └─ 否 → 保留在 conversations/{date}/
```

---

## 💡 最佳实践

1. **及时归档**: 对话结束后立即整理文件
2. **添加说明**: 为每个文件添加简短说明
3. **避免重复**: 检查是否已有类似代码
