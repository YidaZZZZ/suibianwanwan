# AA-LCR 评测项目规划

## 📋 项目概述

**评测基准：** AA-LCR (Artificial Analysis Long Context Reasoning)
**评测对象：** 意识AI Agent（包含逻辑推理引擎）
**评测日期：** 2026年3月31日
**预计成本：** ~27 CodeBuddy积分（$26.06）

---

## 🎯 评测目标

1. **验证长上下文逻辑推理能力**
   - 评估Agent处理300万token文档的能力
   - 测试跨文档综合推理能力

2. **对比分析**
   - 基础模型 vs 改进版Agent
   - 量化架构提升效果

3. **榜单冲击准备**
   - 获取权威评测数据
   - 为后续冲榜做准备

---

## 📊 评测规格

| 参数 | 数值 |
|------|------|
| 文档数量 | 30个文档 |
| 文档总token | ~3,000,000 tokens |
| 问题数量 | 100道 |
| 任务类型 | 跨文档综合推理 |

## 💰 成本预算

| 阶段 | 预计tokens | 美元成本 | 积分 |

## 🚀 执行流程

### 1. 环境准备
```bash
# 安装EvalScope
pip install evalscope[all]

# 验证安装
evalscope --version
```

### 2. Agent封装
- 将Agent封装为可调用服务
- 确保API兼容EvalScope接口
- 配置超时和重试机制

### 3. 运行评测
```bash
evalscope run \
  --dataset evalscope/AA-LCR \
  --model your_agent_name \
  --output_dir ./aa_lcr_result \
  --num_gpus 1 \
  --max_new_tokens 500
```

### 4. 结果收集
- 收集100道题的答案
- 收集准确率数据
- 记录推理时间
- 记录token使用量
